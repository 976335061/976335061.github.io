Hexo 搭建博客(一)

Posted on 2019-04-19 | Edited on 2019-04-20 | In Hexo博客搭建之路

参考bilibili视频

基于MAC OS系统，所有以下在macOS Mojave下进行演示

感谢 UP主CodeSheep

###01 安装 Node.js
官方网站

https://nodejs.org

###02 安装 Git(mac用户跳过)
官方网站

https://git-scm.com/downloads

所有东西按照官方的来就好，Git Bash下载下来就相当于Mac中的终端窗口了

###03 安装 Hexo

国内地址比较慢，所以用一下淘宝的镜像源

npm install -g cnpm --registry=https://registry.npm.taobao.org

验证以下安装是否成功

hexo -v

再创建文件夹

makdir blog
cd blog/

安装hexo

sudo hexo init

04 测试

1:hexo clean #清理缓存
2:hexo g     #生成文件
3:hexo s     #运行本地服务器
4:hexo d     #上传到服务器

打开浏览器输入 localhost:4000

本地环境搭建完成
