Hexo 搭建博客(二)

Posted on 2019-04-19

前面介绍了本地环境的搭建，这次部署到远端服务器，(国内也有许多免费的环境)这里我们选择Github
，由于是国外服务器所以访问速度有些慢

01 安装建立 Github账户

创建一个仓库

02 安装git工具

cnpm install --save hexo-deployer-git

03 配置_config.yml 文件

deploy:
    type: git
    repo: https://github.com/976335061/976335061.github.io.git
    branch: master

配置完成后使用

hexo d

部署到github

若出现以下错误

更改配置文件

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

输入自己的github账号和密码即可
成功后如下图

打开浏览器，在地址栏中输入你自己的github仓库地址就好

976335061.github.com
